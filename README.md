# customer_behavior_analysis
Data analytics project showcasing customer behavior analysis using Python , SQL and PowerBi

This project showcases an end-to-end data analytics workflow, including data loading, cleaning, exploratory data analysis (EDA), SQL-based querying, and interactive dashboard creation. The objective is to transform raw data into actionable insights and present them through clear visualizations and reports for business decision-making.

Dataset
Source: Customer sales dataset (CSV format)
Contents:
Customer details
Product information
Sales transactions
Size: Medium-scale structured dataset suitable for analysis

Tools & Technologies
Python (Pandas, NumPy, Matplotlib, Seaborn) – Data loading, cleaning, and EDA
SQL (PostgreSQL / MySQL / SQL Server) – Querying and data analysis
Power BI – Dashboard creation and visualization
Gamma – Presentation (PPT) creation
Jupyter Notebook – Analysis workflow

Project Workflow
1. Data Loading
Imported dataset using Pandas
Verified structure, data types, and missing values
2. Exploratory Data Analysis (EDA)
Summary statistics and distributions
Identified trends, patterns, and outliers
Visualized key relationships using plots
3. Data Cleaning
Handled missing/null values
Removed duplicates
Standardized column formats
Prepared data for analysis
4. SQL Analysis
Loaded cleaned data into relational databases:
PostgreSQL
MySQL
SQL Server
Executed queries for:
Aggregations (SUM, AVG, COUNT)
Filtering and grouping
Business insights (sales trends, top customers, etc.)
5. Dashboard Development
Built an interactive Power BI dashboard
Included:
KPIs (Revenue, Sales, Profit)
Trend analysis
Customer and product insights
Enabled filtering and drill-down capabilities
6. Reporting
Created a structured report summarizing:
Key findings
Business insights
Recommendations
7. Presentation
Designed a professional presentation using Gamma
Focused on:
Problem statement
Approach
Key insights
Visual storytelling

Dashboard Highlights
Interactive filters for dynamic analysis
Visual breakdown of sales by category, region, and time
Identification of top-performing products and customers
Trend analysis for decision-making

Key Results
Improved understanding of sales performance
Identified high-value customers and products
Detected patterns and trends for business optimization
Delivered insights through interactive and visual formats

This project showcases a complete analytics pipeline, combining technical skills (Python, SQL, BI tools) with business insight generation and communication.
